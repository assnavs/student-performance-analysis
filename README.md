# Student Performance Analysis using SQL and Python

This project analyzes student exam performance using MySQL and Python.

## Tools Used
- MySQL (Data storage and querying)
- Python
- Pandas
- Matplotlib

## Steps Performed
1. Imported CSV data into MySQL database.
2. Wrote SQL queries to analyze average scores by gender, test preparation, and parental education.
3. Exported SQL table data to CSV.
4. Visualized insights using Python and Matplotlib.

## Insights
- Students who completed test preparation scored higher in math.
- Gender-based differences observed in average math scores.
- Parental education level influences student performance.
